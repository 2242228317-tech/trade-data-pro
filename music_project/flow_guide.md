# 韵律与Flow设计指南

## 基本信息
- **BPM**: 85（中等速度，适合说唱）
- **拍号**: 4/4拍
- **风格**: Boom Bap / 新派说唱

## 【主歌1】Flow设计

```
凌晨三点 | 还在debug | 代码      (4拍)
咖-啡-杯 | 见底 | 灵感却 | 不掉       (4拍 - 切分节奏)
别人睡-觉 | 我在 | 拼命跑           (4拍)
这条路 | 孤独 | 但我不 | 求饶        (4拍 - 强调"求饶")

Flow技巧：
- 每句第4拍留白，制造呼吸感
- "咖啡杯见底"用三连音增加律动
- "孤独"和"求饶"拖长音，增加情感
```

## 【副歌】Flow设计

```
从零-开始 | 我不-怕-输            (4拍)
一步-一步 | 走向-我的-路           (4拍)
从零-开始 | 我不-认输             (4拍)
AI时-代 | 我要-做-主              (4拍)

Flow技巧：
- 使用Call & Response结构
- "从零开始"重复制造记忆点
- 每句结尾下沉，营造坚定感
```

## 押韵结构

| 位置 | 韵脚 | 示例 |
|------|------|------|
| 主歌1 | -ao | 代码/不掉/跑/求饶 |
| 主歌2 | -an | 源/燃/盘/阶 |
| 副歌 | -u/-u | 输/路/认输/做主 |
| Bridge | -a/-ua | 吧/大/化/划 |

## 情绪曲线

```
主歌1: 压抑但坚定 → 强度60%
副歌: 爆发释放 → 强度90%
主歌2: 更激进 → 强度70%
Bridge: 叙事平静 → 强度50%
最后副歌: 全力释放 → 强度100%
```
