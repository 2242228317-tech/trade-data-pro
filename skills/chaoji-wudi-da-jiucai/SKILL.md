## 📊 核心功能

### 1. 行情数据获取

使用 `scripts/get_stock_data.py` 获取 A 股行情数据：

```bash
python scripts/get_stock_data.py <股票代码> [参数]
```

**支持的数据类型：**
- 实时行情（开盘、收盘、最高、最低、成交量、成交额）
- 历史 K 线（日 K、周 K、月 K）
- 分时数据
- 资金流向
- 龙虎榜数据

**数据来源：** 新浪财经、东方财富、腾讯财经等免费 API

### 2. 技术分析

使用 `scripts/technical_analysis.py` 计算技术指标：

**支持的指标：**
- 均线系统：MA5、MA10、MA20、MA60、MA120、MA250
- MACD（平滑异同移动平均线）
- KDJ（随机指标）
- RSI（相对强弱指标）
- BOLL（布林带）
- VOL（成交量指标）
- OBV（能量潮）

**分析输出：**
- 金叉/死叉信号
- 超买/超卖状态
- 趋势判断（上升/下降/盘整）
- 支撑位/压力位

### 3. 基本面分析

使用 `scripts/fundamental_analysis.py` 分析公司基本面：

**分析维度：**
- 财务指标（PE、PB、ROE、毛利率、净利率）
- 成长能力（营收增长率、净利润增长率）
- 估值水平（与行业对比、历史分位）
- 股东结构（机构持股、北向资金）

### 4. 风险评估

使用 `scripts/risk_checker.py` 评估投资风险：

**风险因素：**
- 波动率分析
- 最大回撤
- 估值风险（高估/低估）
- 行业风险
- 政策风险提醒

## 🚨 风险提醒规则

以下情况必须主动提醒用户风险：

1. **高位股** - 股价处于历史高位 80% 分位
2. **高估值** - PE 高于行业平均 50% 以上
3. **业绩下滑** - 连续两季度营收/利润负增长
4. **资金流出** - 连续 5 日主力净流出
5. **技术破位** - 跌破重要支撑位/均线

---

## 💡 使用技巧

1. **组合使用** - 技术面 + 基本面结合分析更准确
2. **多看少动** - 分析后不一定要操作，等待高确定性机会
3. **分散投资** - 不要重仓单只股票
4. **定期复盘** - 每周/每月回顾持仓表现
