# SiliconFlow 配置指南

## 1. 注册账号
1. 访问 https://siliconflow.cn/
2. 点击右上角"登录"或"注册"
3. 用手机号注册（最简单）
4. 完成实名认证（如果需要）

## 2. 获取 API Key
1. 登录后，点击右上角头像
2. 选择 "API 密钥"
3. 点击 "新建 API 密钥"
4. 复制密钥（格式：sk-xxxxxxxx）

## 3. 充值（新用户有免费额度）
1. 点击右上角头像 → "账户余额"
2. 新用户通常有赠送额度
3. 如需充值：点击"充值" → 选择金额 → 微信/支付宝支付
4. 建议先充 **¥10-20** 测试

## 4. 配置到本地
把 API Key 粘贴到 `workspace/.env` 文件：
```
SILICONFLOW_API_KEY=sk-你的key
```
