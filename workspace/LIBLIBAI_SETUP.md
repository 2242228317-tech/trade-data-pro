# LiblibAI 配置指南

## 1. 注册账号
1. 访问 https://www.liblib.art/
2. 点击右上角"登录/注册"
3. 用手机号注册
4. 完成实名认证

## 2. 获取 API Key
1. 登录后，点击右上角头像
2. 选择 "开发者" 或 "API"
3. 创建 API Key
4. 复制保存

## 3. 充值
1. 头像 → "账户"
2. 点击"充值"
3. 微信/支付宝支付
4. 建议先充 ¥10-20

## 4. 配置到本地
把 API Key 粘贴到 `workspace/.env`：
```
LIBLIBAI_API_KEY=你的key
```
