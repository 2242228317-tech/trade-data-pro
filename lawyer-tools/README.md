# 律师工具箱 - Legal Tools Pro

为法律专业人士打造的智能工具集，提高办案效率，减少重复工作。

## 🚀 快速开始

### 方法一：直接打开
直接在浏览器中打开 `index.html` 文件即可使用。

### 方法二：本地服务器（推荐）
```bash
# 使用 Python
cd lawyer-tools
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080

# 然后访问 http://localhost:8080
```

## 🛠️ 功能列表

### 计算工具
| 工具 | 功能说明 |
|------|----------|
| 💰 诉讼费计算器 | 根据《诉讼费用交纳办法》计算案件受理费 |
| 📈 利息计算器 | 单利、复利、LPR 利息、逾期利息计算 |
| 📅 工作日计算器 | 计算期间、期限，排除周末 |
| 💵 赔偿金计算器 | 工伤、交通事故、劳动补偿金计算 |

### 文书工具
| 工具 | 功能说明 |
|------|----------|
| 📄 法律文书生成 | 起诉状、答辩状、代理词、律师函等 |
| 📋 文书模板库 | 6 种常用法律文书模板 |

### 管理工具
| 工具 | 功能说明 |
|------|----------|
| 📁 案件管理 | 跟踪案件进度、管理重要日期 |
| ⏱️ 时间追踪 | 记录工作时长，生成计费报表 |
| 🔍 法条检索 | 本地常用法条快速查询 |

## 📋 文书模板

1. **民事起诉状** - 标准民事案件起诉格式
2. **民事答辩状** - 被告答辩标准格式
3. **代理词** - 律师代理意见模板
4. **律师函** - 正式律师函格式
5. **合同模板** - 常用合同范本
6. **授权委托书** - 诉讼代理授权格式

## 💾 数据存储

- 所有数据存储在浏览器本地存储（LocalStorage）
- 数据不会上传到服务器，保证隐私安全
- 支持案件数据导出为 CSV 文件
- 支持法律文书下载为 TXT 文件

## 📊 案件管理

案件管理功能支持：
- 添加/编辑/删除案件
- 案件分类（民事、刑事、行政、仲裁、咨询）
- 客户信息管理
- 重要日期提醒
- 数据导出备份

## ⏱️ 时间追踪

时间追踪功能支持：
- 实时计时器
- 手动添加工作记录
- 按日/周/月统计
- 工作时长汇总报表

## ⚠️ 免责声明

1. 本工具提供的计算结果仅供参考，具体金额以法院核定或相关部门确认为准
2. 法律文书模板需要根据具体案情修改使用，不建议直接套用
3. 法条检索为本地简易数据库，完整法条请查阅官方法律法规数据库
4. 使用本工具不构成律师 - 客户关系，不替代专业法律意见

## 🔧 技术栈

- HTML5 + CSS3 + JavaScript (纯前端，无后端依赖)
- Google Fonts - Noto Sans SC 中文字体
- LocalStorage - 本地数据存储
- 响应式设计 - 支持手机、平板、电脑
